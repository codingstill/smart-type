smartType examples

Open your console to see the messages. The only depedency is jQuery. Bootstrap and Highlightjs are used to style the demo page.

Basic example


    // Set some initial data
    jQuery('.example1').val('Basic example');
    // Initialize textbox
    jQuery('.example1').smartType({
        onSave: function (prev, current, textbox) { console.log('save: ' + current); },
        onCancel: function (prev, current, textbox) { console.log('cancel'); }
    });

Clear on cancel

Type any data and hit ESC 


    // Initialize textbox
    jQuery('.example2').smartType({
        clearOnCancel: true,
        onSave: function (prev, current, textbox) { console.log('save: ' + current); },
        onCancel: function (prev, current, textbox) { console.log('cancel'); }
    });

Require on save

Hit Enter without in empty textbox and see what happens. Then add some data and try again. 


    // Initialize textbox
    jQuery('.example3').smartType({                             
        requireOnSave: true,
        onSave: function (prev, current, textbox) { console.log('save: ' + current); },
        onCancel: function (prev, current, textbox) { console.log('cancel'); }
    });
